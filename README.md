# Last Girl Survivor (라스트걸 서바이버)

**대역전의 종소리가 울린다! 기이한 재판소에서 살아남으세요.**

## 🎮 게임 개요
`Last Girl Survivor`는 긴박한 액션과 전략적인 성장이 결합된 서바이벌 로그라이크 게임입니다. 수학여행 중 눈을 떠보니 도착한 기이한 재판소. 사방에서 조여오는 기계 인형들의 공격 속에서 당신은 살아남아야 합니다.

## ✨ 주요 특징
- **2.5D 그래픽 연출**: 깊이감 있는 캐릭터 그림자와 광원 효과로 구현된 몰입감 넘치는 전장.
- **다양한 무기 및 시너지**: 분필, 리코더, 스마트폰 등 일상적인 아이템을 강력한 무기로 조합하고 숨겨진 시너지를 발동시키세요.
- **영구적 성장 시스템**: 스테이지에서 획득한 뱃지를 통해 기본 능력치를 강화하고 새로운 의상을 해금하세요.
- **리드미컬한 액션**: 박자에 맞춘 '퍼펙트 대쉬'로 위기를 기회로 바꾸는 쾌감을 느껴보세요.
- **오프라인 플레이 지원**: 언제 어디서나 데이터 걱정 없이 즐길 수 있는 100% 오프라인 모드 제공.

## 🕹️ 조작 방법
- **이동**: `WASD` 또는 `방향키` (모바일: 가상 조이스틱)
- **대쉬**: `SHIFT` 키 (모바일: DASH 버튼) - 박자에 맞춰 사용 시 강력한 공격 효과와 쿨타임 초기화!
- **일시정지**: `ESC` 키

## 🛠️ 기술 스택
- **Engine**: GDevelop 기반 커스텀 엔진
- **Platform**: Web, Android (Capacitor)
- **Framework**: AppIntos (AIT) 통합 지원

## 📦 설치 및 실행
1. 안드로이드 기기에서 `PROJ_ISUL_OFFLINE_V[버전].aab` 파일을 통해 설치하거나, 지원되는 웹 브라우저에서 즉시 실행 가능합니다.
2. 최신 버전은 [GitHub Pages](https://kdw319.github.io/ksurvivor/)에서 확인하실 수 있습니다.
